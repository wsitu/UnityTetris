# UnityTetris
A quick implementation of a simple Tetris using Unity and its default assets. 

## Controls

### WASD

|Key  |Action     |
|:---:|-----------|
|W    | rotate    |
|A    | move left |
|S    | move down |
|D    | move right|
|Space| drop down |

### Arrows

|Key  |Action     |
|:---:|-----------|
|Up   | rotate    |
|Left | move left |
|Down | move down |
|Right| move right|
|Space| drop down |

Press any key after the game over animation to restart.
